# 🎉 Sistema Festa & Cia

Sistema simples de gerenciamento de eventos desenvolvido em PHP com MySQL e Bootstrap.

## 📌 Funcionalidades

- Cadastrar eventos
- Listar eventos
- Editar eventos
- Excluir eventos
- Upload de imagem (salva em pasta)

---

## ⚙️ Como rodar o projeto

1. Baixe e instale o XAMPP

2. Abra o XAMPP e inicie:
   - Apache
   - MySQL

3. Clique em "Admin" no MySQL (phpMyAdmin)

4. Crie ou selecione um banco de dados

5. Vá na aba **Importar** e selecione o arquivo `.sql` do projeto

6. Extraia o projeto e coloque a pasta em:

C:\xampp\htdocs\

7. No navegador, acesse:

8. ---

## 🛠️ Tecnologias utilizadas

- PHP
- MySQL
- Bootstrap

---

## 📷 Observação

As imagens dos eventos são armazenadas na pasta `/img` e não diretamente no banco de dados.
